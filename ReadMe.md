## About Me

I build data-driven, production-grade web applications with a focus on AI integration and cloud-native architecture. I work across the full stack — from designing typed APIs and resilient backends to shipping polished frontends with CI/CD pipelines.

- LLM-powered features: retrieval, embeddings, prompt orchestration, and evaluation loops
- Scalable backends with observability, caching, and performance-first design
- Cloud-native delivery with Docker, Kubernetes, and GitHub Actions
